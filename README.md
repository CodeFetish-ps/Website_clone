# Nike Clone — Frontend Project

A pixel-inspired Nike homepage clone built with **pure HTML, CSS, and vanilla JavaScript**. No frameworks, no build tools — just open `index.html` in your browser.

## ✨ Features

- Responsive navigation with cart badge
- Full-screen hero section with animated background
- Scrolling marquee banner
- Interactive category cards with hover effects
- Product grid with color swatch switcher
- "Just Do It" CTA banner
- Nike Membership section
- Footer with links
- Scroll reveal animations
- Mobile responsive

## 🛠 Tech Stack

| Tech | Used For |
|------|----------|
| HTML5 | Structure |
| CSS3 | Styling, animations, grid/flexbox |
| Vanilla JS | Scroll reveal, interactivity |
| Google Fonts | Barlow Condensed + Barlow |

## 🚀 Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/YOUR_USERNAME/nike-clone.git
   ```
2. Open `index.html` in your browser — that's it!

## 📁 Project Structure

```
nike-clone/
└── index.html    # All code in one file (HTML + CSS + JS)
```

## 📤 How to Push to GitHub

```bash

git branch -M main

```
